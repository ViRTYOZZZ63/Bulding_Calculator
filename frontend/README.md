# BuildFlow Frontend (React)

## Быстрый старт
```bash
cd frontend
python3 -m http.server 3000
```

Открой: `http://localhost:3000`

> Для работы API backend должен быть запущен на `http://localhost:8080`.

## Что умеет интерфейс
- Главная с KPI и сценариями работы платформы.
- Калькулятор: создание расчёта, расчёт каркаса и фундамента.
- Материалы: загрузка с backend, поиск и добавление в корзину.
- Личный кабинет: история расчётов и корзина закупки.

## Проверка ошибок
1. Открой DevTools (`F12`) → Console.
2. Обнови страницу, пройди по всем вкладкам.
3. Убедись, что нет красных JS-ошибок.
4. Если API недоступен — увидишь понятные сообщения в интерфейсе.
